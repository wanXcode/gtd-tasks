# 🌸 GTD 使用指南（给 wife）

嗨 wife～这是你的个人任务管理系统，帮你轻松管理工作生活！

---

## 📱 怎么用？

**超级简单**，直接跟我聊天就行：

### 1️⃣ 添加任务
```
帮我记个事：明天下午 3 点牙医预约
```
→ 我会记入你的收集箱

### 2️⃣ 查看待办
```
我今天要做什么？
```
→ 我给你列清单

### 3️⃣ 标记完成
```
牙医预约完成了
```
→ 我帮你归档

### 4️⃣ 查看进度
```
这周完成了什么？
```
→ 我给你总结

---

## 🎯 任务分类（四象限）

我会帮你把任务分到 4 类：

|  |  |  |
|---|---|---|
| 🔴 **Q1** | 紧急重要 | 立即做 |
| 🟡 **Q2** | 重要不紧急 | 计划做 |
| 🟠 **Q3** | 紧急不重要 | 委托做 |
| ⚪ **Q4** | 不紧急不重要 | 少做 |
